# Cookie Empire Clicker

A classic browser clicker game with progression loops:

- Manual clicking with combo and critical hits
- Upgrade shop with scaling costs and caps
- Auto-generating units (CPS economy)
- Achievements and milestone feedback
- Ascension (prestige) with permanent Golden Chips bonus
- Autosave, manual save, reset, and offline progress gains

## Run

Open `index.html` in any modern browser.

No build step required.
